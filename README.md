## Тестовое задание

Тут небольшой исходный код небольшого экрана приложения-конвертера валют.
Экран написан без реактивных фреймворков, на простых замыканиях.

Функционал экрана:
1. Пользователь вводит сумму в текстовое поле (в USD).
2. Выбирает целевую валюту (EUR или GBP) в переключателе (Segmented Control).
3. Нажимает кнопку "Convert".
4. Приложение имитирует сетевой запрос за курсом, показывает индикатор загрузки, а затем отображает результат или ошибку.

Задача кандидату:
Обработать тестовый сценарий:

Допустимые ограничения: от 0 до 100 000. Нужно реализовать ввод тестовой суммы, тап на Convert, запись в лог времени ответа. Затем изменение с EUR на GBP, повторный тап на Convert, запись в лог времени ответа. На выходе нужно этот лог сохранить в формате json.

## Где найти JSON лог

- Во время UI теста файл c результатами сохраняется в временную папку раннера, точный путь печатается в логах: `JSON log saved at: ...`.
- Также JSON прикладывается как attachment conversion_response_log.json в Test Report.
